# 📖 About The Project

SmartKart is a modern e-commerce frontend application developed using **React.js** and powered by **REST API Integration**.

The project demonstrates how to build a scalable shopping interface with reusable React components, dynamic product rendering, intelligent filtering, real-time searching, and responsive layouts.

The primary goal of this project is to showcase modern frontend development skills while maintaining clean architecture, optimized performance, and an excellent user experience.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/AbdulRafay901/SmartKart-react.js.git
```

---

## Navigate into Project

```bash
cd SmartKart-react.js
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Build Production Version

```bash
npm run build
```
